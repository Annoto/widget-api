### `Optional` showReplies

• **showReplies**? : *boolean*

If set to true, when widget is loaded all the replies to comments will be shown.

___

### `Optional` sort

• **sort**? : *[SortByType](../modules/annotoconfig.md#sortbytype)*

The default sorting to use on startup.
Default: 'most_recent'
